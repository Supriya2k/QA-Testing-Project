# QA Playwright Automation Portfolio

## Overview
This project demonstrates QA Automation skills using Playwright.

## What is included
- UI Automation (Login, Cart flow)
- API Testing
- Page Object Model (POM)
- Test design documentation

## Tech Stack
- Playwright
- TypeScript

## Project Structure
- tests/ui → UI automation tests
- tests/api → API tests
- pages → Page Object Model
- docs → QA documentation

## Note
This project was developed using a mobile environment and structured for Playwright execution in a standard setup.
